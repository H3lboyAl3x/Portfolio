## 🚀 Tecnologias Utilizadas

* **Estrutura:** HTML5 Semântico.
* **Estilização:** CSS3 Avançado.
* **Interatividade:** JavaScript Vanilla.
* **Efeitos Visuais:** HTML5 Canvas API.
* **Design:** Conceitos *Tecnologico*.

## 🧠 Estrutura de Navegação

O portfólio foi arquitetado em módulos para facilitar a conversão do visitante:
1.  **Início:** Landing page de alto impacto com animações de flutuação e tipografia neon.
2.  **Sobre:** Dashboard detalhado com foco em Stack Tecnológica, Banco de Dados e Soft Skills.
3.  **Projetos:** Grid dinâmico para exibição de soluções reais com links diretos para repositórios e demonstrações.
4.  **Contatos:** Central de comunicação direta com links sociais integrados em uma interface tipo terminal.

## 🎯 Objetivo do Projeto

Este projeto demonstra competências técnicas em:
* Criação de interfaces complexas e exclusivas sem dependência de bibliotecas externas.
* Manipulação avançada da **Canvas API** para animações baseadas em física e proximidade.
* Organização de código escalável e aplicação de boas práticas de UI/UX.
